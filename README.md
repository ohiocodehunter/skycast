# 🌦️ SkyCast Weather App

A modern, responsive weather web application built using **HTML, CSS, and JavaScript**.
This project provides real-time weather data along with an interactive UI, animated backgrounds, and a dynamic bear assistant.

---

## 🚀 Features

* 📍 **Auto Location Detection** using browser geolocation

* 🔍 **Search by City** (press Enter to fetch weather)

* 🌡️ **Current Weather Details**

  * Temperature
  * Weather condition
  * Humidity, wind speed, feels like

* 📊 **Hourly Forecast**

  * Scrollable hourly weather (like mobile apps)

* 📅 **7-Day Forecast**

  * Clean and responsive layout

* 🌗 **Auto Day/Night Theme**

  * Changes UI based on local time

* 🌦️ **Animated Background Effects**

  * Rain animation
  * Sun glow
  * Clouds & night stars

* 🐻 **Interactive Bear Assistant**

  * Fully animated SVG bear
  * Reacts to weather conditions
  * Provides helpful suggestions

* ⚡ **Smooth Animations**

  * Powered by GSAP

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Glassmorphism + Responsive Design)
* JavaScript (Vanilla JS)
* GSAP (Animations)
* WeatherAPI (Weather Data)

## 🔑 Setup Instructions

1. Clone or download this repository
2. Open `script.js`
3. Replace the API key:

```js
const API_KEY = "YOUR_API_KEY_HERE";
```

4. Open `index.html` in your browser
